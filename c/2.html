ఇక్కడ మనం ఒక బేసిక్ C ప్రోగ్రాం యొక్క స్ట్రక్చర్ చూద్దాం. 

#include <stdio.h>

int main() {
    //print on screen
    printf("Hello, World!");
    return 0;
}

వీటిని కామెంట్స్ అంటారు. ప్రోగ్రాం గురుంచి వేరే వారికి తెలియజేయటం కోసం వాడతాం. ఒక లైన్ లో గాని, ఎక్కువ లైన్ లలో గాని వాడవచ్చు.

#include <stdio.h>
దీనిని preprocessor directive అంటారు. ఇక్కడ stdio అంటే స్టాండర్డ్ ఇన్పుట్ ఔట్పుట్. అంటే కీబోర్డ్ నుండి ఇన్పుట్, మానిటర్ లో ఔట్పుట్ అన్నమాట. మనం మన ప్రోగ్రాం లో కీబోర్డ్ మరియు మానిటర్ ఉపయోగించాలి కాబట్టి ప్రోగ్రాం లో ఇది వాడతాం.

int main() {
}
int main అని ఉంది కదా? ప్రతి సి ప్రోగ్రాం యొక్క స్టార్టింగ్ పాయింట్ ఇది. అంటే ప్రోగ్రాం రన్ అవటం ఇక్కడ నుండి మొదలవుతుంది. తర్వాత {} braces గమనించండి. ఓపెన్ అండ్ క్లోజ్. ప్రతి ఫంక్షన్ కి ఉంటాయి. ఇక్కడ మెయిన్ ఒక ఫంక్షన్. బ్రేసెస్ ఇంకా ఎక్కడ వాడుతాం అనేది రాను రాను చూస్తాం.
