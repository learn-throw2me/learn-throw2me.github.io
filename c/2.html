Basic Structure

ఇక్కడ మనం ఒక బేసిక్ C ప్రోగ్రాం యొక్క స్ట్రక్చర్ చూద్దాం. 

#include <stdio.h>

int main() {
    //print on screen
    printf("Hello, World!");
    return 0;
}

// comments
/* comments
in
multiple
lines */
వీటిని కామెంట్స్ అంటారు. ప్రోగ్రాం గురుంచి వేరే వారికి తెలియజేయటం కోసం వాడతాం. ఒక లైన్ లో గాని, ఎక్కువ లైన్ లలో గాని వాడవచ్చు.

#include <stdio.h>
దీనిని preprocessor directive అంటారు. ఇక్కడ stdio అంటే స్టాండర్డ్ ఇన్పుట్ ఔట్పుట్. అంటే కీబోర్డ్ నుండి ఇన్పుట్, మానిటర్ లో ఔట్పుట్ అన్నమాట. మనం మన ప్రోగ్రాం లో కీబోర్డ్ మరియు మానిటర్ ఉపయోగించాలి కాబట్టి ప్రోగ్రాం లో ఇది వాడతాం.

int main() {
}
ప్రతి C ప్రోగ్రాం యొక్క స్టార్టింగ్ పాయింట్ ఇది. అంటే ప్రోగ్రాం రన్ అవటం ఇక్కడ నుండి మొదలవుతుంది. తర్వాత braces {} గమనించండి. ఓపెన్ అండ్ క్లోజ్. ప్రతి ఫంక్షన్ కి ఉంటాయి. ఇక్కడ మెయిన్ ఒక ఫంక్షన్. బ్రేసెస్ ఇంకా ఎక్కడ వాడుతాం అనేది రాను రాను చూస్తాం.

printf("Hello, World!");
ఇది ఒక ఫంక్షన్. ఇందాక మెయిన్ అని ఒక ఫంక్షన్ అని ఎలా చెప్పుకున్నామో, ఇది కూడా ఒక ఫంక్షన్. ఈ ఫంక్షన్ వాడాలంటే మనం పైన వాడిన stdio preprocessor directive వాడాలి. Parantheses () లోపల "Hello , World!" అని ఉంది. అంటే మానిటర్ మీద ఆ వాక్యం రావాలి అని మనం ప్రోగ్రాం చేస్తున్నాం. Double quotes " ", చివర్లో semicolon ; చాలా ముఖ్యం.

return 0;
ప్రస్తుతం ఇది ఒక రూల్ లాగ గుర్తుంచుకోండి. మనం ఫంక్షన్స్ టాపిక్ లో దీని గురించి వివరంగా తెలుసుకుందాం.

పైన ప్రోగ్రాం, ఈ క్రిందది ఒకటే. Open brace పైన ఒకే లైన్ లో ఉంది. క్రింద కొత్త లైన్ లో మొదలైంది. మీ వీలుబట్టి ఎదో ఒకటి ఫాలో అవ్వండి. 

int main() 
{
    //print on screen
    printf("Hello, World!");
    return 0;
}
